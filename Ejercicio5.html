Investigación de la siguientes propiedades

Propiedad Visibility

Esta propieda tiene dos efectos:

Hidden: que esconde el elemento, sin embargo su espacio queda vacío

Collapse: que esconde filas o columnas de una tabla

Y visible que se aplica a todos los elementos

Visibility nos dice si un elemento es visible o no

Propiedad Display

Esta propiedad nos ayuda a identificar visualmente interna y externa de un elemento

Existe display: none; que sirve para ocultar y mostrar elementos sin eliminarlos, entre otros valores como inline y block.

El Display:inline-block es diferente porque es una caja, pero se comporta como si fuera un bloque

Display:Block: este va mostrar los elementos como un block, siempre va tener un espacio ya sea encima o por debajo, que lo separa de los otros elementos.

Propiedad Overflow

Esta propiedad nos da la flexibilidad de poder controlar el comportamiento del contenido que se encuentra en una caja o contenedor podemos especifica si queremos recortar,dibujar barras de desplazamient, o mostrar el contenido se excede.

Sus Valores son:

Overflow: visible: ESte lo que hace es que los contenidos puedan sea visibles y se salgan del elemento.

Overflow: hidden: con este se tiene control del tamaño y contenido del elemento.

Overflow: scroll: Cuando los contenidos no caben en el contenedor, este muestra una barra horinzontal y vertical

Overflow: auto: como depende el navegador se va visualizar de forma diferente, este valor ayuda para eso no interfiera para que el contenido no se desborde del contenedor y si en un dado caso se sale del lado horizontal o vertical, solo va aparecer la barra de scroll.
